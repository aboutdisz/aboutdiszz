# 🔥 MY TOOLS - IP & Phone Tracker


Tools sederhana untuk melacak IP dan nomor telepon langsung dari Termux.

---

## 📌 FITUR
- 🎯 **Track IP Address** - Cari tahu lokasi, ISP, dan koordinat IP target
- 📱 **Track Nomor Telepon** - Cek negara, operator, dan zona waktu
- 💻 **Mudah Digunakan** - Interface sederhana, tinggal pilih menu
- ⚡ **Cepat & Ringan** - Tanpa database, langsung pake API

---

## 📦 INSTALLASI

### 1. Clone repository
```bash
git clone https://github.com/aboutdisz/aboutdisz.git
```

### 2. Masuk ke folder
```bash
cd aboutdisz
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Jalankan tools
```bash
python main.py
```

---

## 📝 CARA PENGGUNAAN

### Track IP
1. Pilih menu **[1]**
2. Masukkan IP target (contoh: `8.8.8.8`)
3. Hasil akan muncul

### Track Nomor Telepon
1. Pilih menu **[2]**
2. Masukkan nomor dengan kode negara (contoh: `+6281234567890`)
3. Hasil akan muncul
